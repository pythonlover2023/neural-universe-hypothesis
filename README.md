## What is this?

A collaborative research project testing whether the universe exhibits neural-network-like information processing dynamics.

## Current Hypothesis (v1.2)

*"The universe exhibits self-organizing information-processing dynamics mathematically equivalent to neural network optimization under information-theoretic constraints."*

Key claims:

- Dark matter as "latent space" representation
